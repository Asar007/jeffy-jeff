Critical Issues — Fix This Week

CRITICAL 1Four of five navigation links return 404

/about, /projects, /customization, and /contact all return HTTP 404. Every procurement persona abandons the site at the first click. The hero "Start a Conversation" CTA and all five footer social icons also route to /contact — every conversion path is dead.

Fix: Deploy the inner pages, or as a same-day stopgap, redirect those routes to homepage anchors (/#about, /#projects, etc.). Replace the contact CTA with a working mailto:md@madrasswastic.com link immediately.

CRITICAL 2Canonical domain is a random Vercel slug

rocket-ship-plum.vercel.app has zero brand signal, zero backlink equity, and could be reassigned if the Vercel project is renamed. AI engines (ChatGPT, Perplexity, Google AI Overviews) cluster authority around named brand domains — there is currently no path to being cited for "Madras Swastic Engineers."

Fix: Register madrasswastic.com (or .in) today. In Vercel → Project → Domains, attach the custom domain and mark it primary. Vercel auto-301s from the slug. Update every canonical, og:url, and schema @id to the new domain.

CRITICAL 3/robots.txt and /sitemap.xml both return 404

AI crawlers (GPTBot, ClaudeBot, PerplexityBot) and Google have no discovery path and no policy signal. The absence of a sitemap also means no clear page inventory once inner pages exist.

Fix: Publish both at the root.

No Organization, no LocalBusiness, no BreadcrumbList, no Service. For a 27-year ISO-certified contractor with ISRO as a named client, this is the highest-leverage missing signal — every other SEO improvement compounds off this baseline.

CRITICAL 5No RFQ / vendor pre-qualification path

The only CTA leads to a 404. No quote form, no downloadable capability brochure, no ISO certificate PDF, no project portfolio. B2B procurement teams cannot self-qualify the vendor and will not email blindly.

Fix: Build /rfq with structured fields (Project type [Structural Fab / Plant Erection / Piping / Turnkey], Tonnage estimate, Location, Timeline, Message). Publish the ISO 9001:2015 certificate as a downloadable PDF. Add a /capability-brief.pdf with crane tonnage, weld capabilities, equipment list.

High-Priority Issues

HIGH 6H1 contains zero target keywords

Current H1: "Engineering Precision for Modern Infrastructure" — brand copy, not a search signal. None of the four target queries appear.

Fix: H1 → "Structural Fabrication & Plant Erection Contractor, Chennai — Madras Swastic Engineers". Title tag → "Structural Fabrication Contractor Chennai | ISO 9001 | Madras Swastic Engineers".

HIGH 7No meta description, no canonical, no Open Graph tags

Social shares produce generic previews. Duplicate-URL signals between the Vercel slug and any future custom domain are uncontrolled. Twitter/X cards do not render.

Fix: Add unique 150–160 char meta description + canonical + full OG/Twitter card metadata on every page.

HIGH 8No dedicated service / capability pages

Each capability (Structural Fabrication, Mechanical Erection, Piping & Skid, Turnkey) maps to a distinct high-intent procurement query, but there is no dedicated page for any of them. Competing vendor sites run 2,500–4,000 words across capability and portfolio sections.

Fix: Build four service pages, 800–1,200 words each, with material grades (IS 2062, ASTM A36), plate thickness range, weld procedure specs (GTAW/SMAW), NDT capabilities, equipment list (crane tonnage, transport reach), and one named project reference per page.

HIGH 9Marquee client claims are asserted, not evidenced

ISRO, Schwing Stetter, and Manali Petrochemicals logos with no project case studies, no scope (tonnage, timeline, location), no client-attributed quote. Logos alone score poorly with both Google E-E-A-T evaluators and AI citation engines.

Fix: One 300-word case study per anchor client, formatted as: "ISRO LPSC — 340 MT structural package, delivered 11 days ahead of schedule, August 2023." Include client logo with descriptive alt text.

HIGH 10ISO 9001:2015 stated as text but never linked

No certificate number, no expiry date, no PDF download, no registrar verification link. Multinational procurement compliance teams (the kind that vet ISRO vendors) cannot proceed past this gap.

Fix: Publish the certificate PDF; add hasCredential with the registration number in schema; link from a homepage credibility bar.

HIGH 11No named principal / director bio

B2B industrial vendors need accountable humans. The email md@madrasswastic.com exists but no MD name, no engineering qualifications, no GST number, no CIN. Enterprise procurement teams running structured vendor qualification cannot complete this step.

Fix: About page with named MD, engineering credentials (AMIE / B.E. Mech / M.Tech), GST number, CIN (from MCA), and ISO certificate registration number.
